# GetSongBPM API Credit

This project uses the [GetSongBPM API](https://getsongbpm.com) for music metadata (BPM, key, time signature).

**Required Attribution:**
GetSongBPM provides free API access with the requirement of a backlink to https://getsongbpm.com

**Where to add the backlink:**
- Project README
- Public-facing website (bopbot.live when launched)
- Application store listing if applicable

**License Terms:**
Using their API is free but a link back to GetSongBPM.com is REQUIRED (website or store listing), or they will suspend your account without notice.
